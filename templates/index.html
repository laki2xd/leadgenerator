How it works

  • Search by Industry: Enter an industry name (e.g., "Manufacturing", "Distribution", "Warehouse", "Logistics")
  • Search by Product: Enter a product/object name (e.g., "Steel pipes", "Automotive parts") to find manufacturers
  • Click "Search Companies" to find B2B companies that need transportation services
  • Automatically filters out: Retail stores, restaurants, cafes, small shops
  • Focuses on: Manufacturing, warehouses, distribution centers, logistics companies
  • View contact information including phone numbers and email addresses (when available)
  • Export results to Excel for lead generation and sales prospecting

Setup Required

To use this application, you need to set up API keys:

  1. Google Places API: Get your key from Google Cloud Console
  2. Yelp API (Optional): Get your key from Yelp Developers
  3. Set environment variables: GOOGLE_PLACES_API_KEY and YELP_API_KEY
